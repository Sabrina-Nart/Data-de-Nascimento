# Data-de-Nascimento
Exercício solicitado durante as aulas de Programação Para Plataformas Móveis, seguindo os requisitos:
Desenvolva uma página web que o usuário informe uma data de nascimento em um
campo e seja apresentado em um parágrafo logo abaixo se ele é:
a. Jovem - Indivíduos de até 19 anos.
b. Adulto - Indivíduos com idade entre 20 até 59 anos.
c. Idoso - Indivíduos de 60 anos em diante.
